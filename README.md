# AutoCode
自动生成C#三层代码框架工具，支持常用的关系型数据库：PostgreSQL、Oracle、MySQL、SqlServer

&nbsp;

### 功能与进度

- [x] 支持PostgreSQL数据库三层生成
- [ ] 支持SQLite数据库三层生成
- [ ] 支持Oracle数据库三层生成
- [ ] 支持MySQL数据库三层生成
- [ ] 支持SQL Server数据库三层生成
- [ ] 自动生成网站后台
- [x] 支持LayerUI框架后台管理界面
- [ ] 支持Bootstrap框架后台管理界面
